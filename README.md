# SSNAnti-Steal
A simple Anti-Steal script for Stormworks

## Documentation

### Commands
#### Users
    • ?c[ID]					              clear players vehicle by ID
    • ?c					                  clear all
    • ?vehicles				              list players vehicles and ID

#### Admins
    • ?vehicles				              list all players vehicles, IDs and players name
    • ?c[ID]					              clear any vehicle by ID
    • ?c					                  clear all vehicles by admin
    • add_player [name]        			add a player to the vehicle list
    • remove_player[name]      			remove a player from the vehicle list
    • add_vehicle [name] [ID]    		add a vehicle to a player in the vehicle list
    • remove_vehicle [name] [ID]		remove a vehicle from a player  in the vehicle list

#### Other stuff
    • [name] supports spaces by entering /&&
